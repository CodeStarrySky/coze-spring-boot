# 示例说明
该示例是航空公司的客户聊天支持代理。

它能够支持已有机票的预订详情查询、机票日期改签、机票预订取消等操作。

> 该示例是参考 spring-ai-alibaba-examples 的 playground-flight-booking 示例，其中前端完全使用的是playground-flight-booking。
>

# 目录介绍
frontend 是前端项目的代码。
